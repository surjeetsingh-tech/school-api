📚 School API

A backend project built using Node.js, Express, and MySQL. This API allows users to manage school
data efficiently with proper CRUD operations.

# Features

* Add new school data
* Get all schools
* Update school details
* Delete school records
* RESTful API structure
* MySQL database integration

## 🛠️ Tech Stack

* Node.js
* Express.js
* MySQL
* JavaScript


## 📂 Project Setup

1️⃣ Clone the repository

git clone https://github.com/surjeetsingh-tech/school-api.git


2️⃣ Install dependencies

npm install
